Nuestros desarrollos para escritorio han sido siempre con tecnologia Microsoft .NET y .NET Core. Desarrollamos y administramos soft para el procesamiento grandes cantidades de datos. Utilizando Visual Basic .Net y C# y Entity Framework para la persistencia. Tambien scritp para procesamientos por largos periodos, interactuando con colas de mensajes y API's

Nuestros desarrollo web se basan en la metodologia Mobile-First para garantizar un resultado responsive que le garantice al cliente que se va a ver igual en cualquier pantalla o dispositivo. API's RestFul con autenticacion mediante Identity, Identity Server y JWT

Tenemos conocimiento y experiancia con distintos motores de bases de datos.
Conocimiento de DAX para generar la información de tableros con Power BI tanto en la nube, como la instalación y administración de servidores de reportes locales. Todos los diseños de los reportes son de manera responsive para porder ser visualizados de la mejor manera en distintos dispositivos.
